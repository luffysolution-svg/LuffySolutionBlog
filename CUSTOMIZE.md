# 模板使用说明

`LuffySolutionBlog` 是 LuffySolution 的个人公开博客，不再作为通用模板发布。

创建自己的博客请使用独立模板仓库：[luffysolution-svg/LSBlogs](https://github.com/luffysolution-svg/LSBlogs)。模板版包含：

- 通用占位配置与示例内容
- 本地可视化博客管理器
- 安全的预览、构建和 Git 发布流程
- Vercel 部署与 Utterances 评论配置说明
- 原项目作者与二次开发者致谢

本仓库中的 `siteConfig.ts`、文章、图片和评论配置属于正式个人站内容，不应直接作为新博客的默认资料复制。
